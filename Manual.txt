[Font]:20,600.Инструкция к приложению FileStruct

[Font]:14,400.Это приложение позволяет просматривать файлы(в основном фотографии) и устанавливать им метки.

[Font]:20,600.Оглавление[Font]:14,400.
1. Виджет для просмотра фотографий.
2. Список фотографий.
3. Список меток.
4. Меню действий.

[Font]:18,600.1. Виджет для просмотра фотографий.
[Font]:14,400.В центре приложения находится место(секция) для просмотра фотографий. С фотографиями в этом поле можно взаимодействовать:
	- Можно приближать и отдалять фотографию при помощи колёсика мыши;
	- можно передвигаться по фотографии при зажатой левой кнопки мыши(ЛКМ);
	- можно повернуть фотографию на 90 градусов вправо или влево при нажатии на соответствующие кнопки справа от фотографии.

[Font]:18,600.2. Список фотографий.
[Font]:14,400.2.1. В нижней левой части находится список фотографий. Каждый элемент списка представляет собой путь до соответствующего файла.
При нажатии левой кнопкой мыши(ЛКМ) по файлу отобразит его в секции просмотра фотографий, а также в списке меток(см. п.3) появятся соответствующие метки. Если файл не является фотографией, то отобразится сообщение об этом.

2.2. Нажатием правой кнопкой мыши(ПКМ) по файлу откроется контекстное меню со следующими действиями:
	- "Добавить метку" - откроется модальное окно, в которое необходимо ввести метку для добавления к файлу. При нажатии кнопки "Ок" метка добавится к файлу. Если значение метки будет пустым, оно не добавится. При нажатии "Отмена" или закрытии окна метка к файлу добавлена не будет.
	- "Добавить существующую метку" - откроется модальное окно, в котором можно будет выбрать метку из списка ранее добавленных меток. Если этот список пуст, то метку можно будет ввести. Кнопки действуют аналогично предыдущему действию. 

2.3. Также над списком файлов находятся кнопки для фильтрации элементов:
	- "Показать все" - отображает все добавленные в приложение файлы;
	- "Отфильтрованные" - (см. п. 4 в вкладке "Фильтрация") отображает файлы, которые получились в результате фильтрации по меткам.
	- "Показать отмеченные" - отображает файлы, у которых есть хотя бы одна метка;
	- "Показать неотмеченные" - отображает файлы, у которых отсутствуют метки.
2.4. При двойном нажатии левой кнопкой мыши(ЛКМ) по элементу списка файлов откроется окно, в котором можно будет просматривать изображения, находящиеся в списке. Оно функционирует так же как и секция просмотра фотографий, и дополнительно позволяет перелистывать файлы стрелками влево и вправо.

[Font]:18,600.3. Список меток.
[Font]:14,400.Также внизу справа находится список меток, относящихся к выбранному файлу.
Нажатием правой кнопкой мыши(ПКМ) по метке в списке меток откроется контекстное меню с возможностью её удаления у выбранного файла.

[Font]:18,600.4. Меню действий.
[Font]:14,400.Вверху приложения находится меню с выбором действий для работы с файлами и метками. 

4.1. Действия в верхнем меню приложения:

[Font]:18,400.Вкладка "Открыть":[Font]:14,400.
	- "Открыть папку" - появляется окно для выбора папки. После выбора все файлы из папки будут загружены в приложение и выведены в список файлов для просмотра. При этом если ранее уже в списке находились файлы, они заменятся на новые из этой папки.
	- "Открыть метки" - появляется окно для выбора файла, в котором сохранены метки. Этот файл должен быть расширения ".ini". После этого файлы и соответствующие им метки будут загружены в приложение и заменят ранее добавленные. Рекомендуется выбирать это действие, если метки уже однажды были сохранены.

[Font]:18,400.Вкладка "Добавить":[Font]:14,400.
	- "Добавить папку" - появляется окно для выбора папки для добавления. После выбора все файлы из папки будут загружены в приложение и добавлены в список, не убирая ранее добавленные.
	- "Добавить метки" - появляется окно для выбора файла, в котором сохранены метки. Этот файл должен быть расширения ".ini". После этого файлы и соответствующие им метки будут загружены в приложение и добавятся в список. Рекомендуется выбирать это действие, если метки уже однажды были сохранены.

[Font]:18,400.Вкладка "Сохранить":[Font]:14,400.
	- "Сохранить метки" - появляется окно для сохранения меток в файл. К нему будет автоматически добавлено расширение ".ini".

[Font]:18,400.Вкладка "Фильтрация":[Font]:14,400.
	- "Фильтрация по меткам" - откроется специальное окно для добавления меток для фильтрации файлов. Те файлы, которые содержат все добавленные метки, пройдут фильтрацию. Чтобы добавить метку необходимо выбрать её из выпадающего списка и нажать кнопку "Добавить", она добавится в список снизу. Чтобы удалить метку нужно выбрать её из списка снизу и нажать кнопку "Убрать". При нажатии кнопки "Ок" окно закроется и сформируется список отфильтрованных файлов. При нажатии "Отмена" или закрытии окна фильтрация отменится.
	- "Копировать отфильтрованные" - откроется окно для выбора папки, в которую необходимо скопировать файлы, отфильтрованные по меткам.
	- "Переместить отфильтрованные" - откроется окно для выбора папки, в которую необходимо переместить файлы, отфильтрованные по меткам.
